Contributing to Bionium-X
=========================

We welcome contributions from the astrophysics and machine learning communities! 

How to Contribute
-----------------
1. Fork the repository on GitHub.
2. Create a feature branch: ``git checkout -b feature/new-physics-model``.
3. Commit your changes with descriptive messages.
4. Push to your branch and submit a Pull Request.

Please ensure that your code adheres to standard `PEP 8` guidelines and includes adequate Pytest coverage.
